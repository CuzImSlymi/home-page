Featured Project: Apertis LLM

A novel, efficient, and simple Multimodal LLM Architecture.

Apertis is designed from the ground up to overcome limitations in existing LLMs, focusing on linear-time processing, memory efficiency, native multimodal support, and ease of use. It integrates cutting-edge concepts into a cohesive and powerful architecture.

View on GitHub

Core Architectural Innovations:

⚙️

Selective Linear Attention (SLA)

Achieves O(n) time complexity for efficient processing of long sequences, adapting attention dynamically.

🧠

Adaptive Expert System (AES)

Inspired by MoE, it dynamically routes tokens to specialized experts for efficient parameter usage and computation.

💾

State Tracking Recurrent Cell (STRC)

Combines RNN-like state tracking with transformer capabilities for efficient inference and context management.

🖼️

Unified Multimodal Encoder (UME)

Seamlessly processes and integrates both text and image data within a joint embedding space.

Professional Training & Interface

Apertis includes a user-friendly, Google AI Studio-inspired UI built with React/FastAPI, featuring YOLO-style real-time training visualization and cross-platform compatibility (Windows, macOS, Linux via Docker).
